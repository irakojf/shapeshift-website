What is cpapacket?

cpapacket is a private internal CLI integration used by Shapeshift to read QuickBooks Online accounting data and generate internal reporting for accounting review and tax preparation.

This integration is not a public product and is not distributed to third parties.

QuickBooks Data Accessed

cpapacket may access the following QuickBooks Online data:

  • Company information and account lists
  • Transactions (for example invoices, expenses, and journal entries)
  • Standard accounting reports, including Profit & Loss, Balance Sheet, and General Ledger

Data is used only for Shapeshift's internal financial reporting workflows. Data is not sold.

How OAuth Connection Works

Authorization is initiated from the cpapacket CLI using Intuit OAuth 2.0. The authorized user signs in to QuickBooks Online and approves access, and cpapacket exchanges the authorization code for API credentials.

Launch URL and Connect/Reconnect URL: https://apps.shapeshift.so/cpapacket/connect. Access can be revoked any time from QuickBooks Online Connected Apps settings.
